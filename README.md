## ✨ What is this?

**Logos Apps** is a curated library of **2,091 SVG logos** — apps, dev tools, frameworks, payment providers, cloud platforms and more. Every file is a lightweight, scalable vector you can drop straight into your website, slide deck, README, design file or side project.

- 🪶 **Lightweight** — pure SVG, no dependencies.
- 📐 **Scalable** — crisp at any size, from a favicon to a billboard.
- 🎯 **Consistent naming** — find any logo in one guess.
- 🆓 **Free to grab** — clone, download, or hotlink.

## 🚀 Quick start

### Option 1 — Download everything

[**⬇️ Download the full pack (.zip)**](https://github.com/ln-dev7/logos-apps/archive/refs/heads/master.zip)

### Option 2 — Clone the repo

```bash
git clone https://github.com/ln-dev7/logos-apps.git
```

### Option 3 — Grab a single logo

Hotlink any SVG directly via its raw URL:

```
https://raw.githubusercontent.com/ln-dev7/logos-apps/master/<name>.svg
```

```html
<!-- Example: GitHub logo -->
<img src="https://raw.githubusercontent.com/ln-dev7/logos-apps/master/github.svg" height="32" alt="GitHub" />
```

---

## 🏷️ Naming convention

| Pattern | Example | Meaning |
| --- | --- | --- |
| `<brand>.svg` | `figma.svg` | The **icon / symbol** only |
| `<brand>-wordmark.svg` | `figma-wordmark.svg` | The **logotype** (icon + name, or text) |

> 💡 There are **448 wordmark** variants. If a brand has both, you'll find `brand.svg` *and* `brand-wordmark.svg`.

## 🤝 Contributing

Got a logo that's missing? PRs welcome!

1. Fork the repo.
2. Add your `brand.svg` (and `brand-wordmark.svg` if relevant) following the naming convention.
3. Keep it optimized — run it through [SVGO](https://github.com/svg/svgo) first.
4. Open a Pull Request 🎉

**Guidelines:** lowercase, hyphenated filenames (e.g. `google-wallet.svg`), no embedded raster images, and trim unnecessary metadata.
